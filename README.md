# Learn-CPP

## Summary

- Xcode command line project to learn and demo modern C++ theory 
  and concepts.
- Implementation is trivial and no User or Developer Guide is needed.
- Build Xcode and run it. Adjust which Demo to see in the `run()` method
  of `Solution.cpp`.
